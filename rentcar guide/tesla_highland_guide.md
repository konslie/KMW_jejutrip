# 🚗 현대차(아이오닉 5) 유저를 위한 테슬라(모델 3 하이랜드) 조작 가이드

본 가이드는 현대차 아이오닉 5 조작 방식에 익숙한 운전자가 테슬라 모델 3 하이랜드(24년식)를 렌트했을 때 당황하지 않도록 모든 핵심 기능을 정리한 최종 본입니다.

## 1️⃣ 필수 주행 및 안전 (Safety & Core Driving)

| 항목 | 현대차 (아이오닉 5) | 테슬라 (모델 3 하이랜드) | 비고 |
| :--- | :--- | :--- | :--- |
| 시동 걸기 | 브레이크 + 시동 버튼 클릭 | 카드키 거치 + 브레이크 밟기 | 별도의 ON/OFF 버튼 없음 |
| 기어 변속 | 핸들 뒤 회전형 레버 | 화면 왼쪽 끝 위(D)/아래(R) 스와이프 | 천장 비상등 옆 물리 버튼 존재 |
| 방향 지시등 | 핸들 왼쪽 레버 조작 | 핸들 왼쪽 물리 버튼 클릭 | 하이랜드 전용: 레버가 아예 없음 |
| 회생 제동 | 패들 시프트로 단계 조절 | 조절 불가 (강한 제동 고정) | 원페달 드라이빙 상시 적용 |
| 비상등 | 대시보드 중앙 버튼 | 천장(백미러 위) 물리 버튼 | 긴급 상황 대비 위치 숙지 필수 |
| 오토 홀드 | Auto Hold 버튼 활성 필요 | 정차 시 브레이크 깊게 밟기 | 테슬라는 상시 활성화가 기본 설정 |

| 와이퍼 조작 | 핸들 오른쪽 레버 조작 | 핸들 왼쪽 버튼 클릭 (1회 작동) | 클릭 시 화면 조절 메뉴 즉시 팝업 |

## 🗺️ 네비게이션 & 스마트폰(아이폰) 연동 팁

**목적지 설정 방법**
* **화면 입력:** 메인 화면 왼쪽 상단의 '어디로 이동할까요?' 검색창을 터치하여 직접 입력합니다.
* **음성 명령:** 핸들 오른쪽의 **음성 명령 버튼(🎙️)**을 누르고 "성산일출봉으로 가줘"와 같이 말하면 가장 빠르고 정확합니다.

**아이폰 연동 및 길 찾기 팁**
* **애플 카플레이 미지원:** 테슬라는 카플레이를 지원하지 않아 티맵/카카오내비를 차량 화면에 띄울 수 없습니다.
* **목적지 전송 기능:** 아이폰 '네이버 지도'나 '구글 지도' 등에서 목적지를 검색한 후, **공유 → Tesla 앱**을 선택하면 차량 네비게이션으로 목적지가 즉시 전송되어 편리합니다.
* **오디오:** 블루투스 연결을 통해 아이폰의 음악(애플뮤직, 유튜브 뮤직 등)을 차량 스피커로 출력할 수 있습니다.

## 🏎️ 원페달 드라이빙 (Hold 모드) 심화 가이드

* **작동 원리:** 가속 페달에서 발을 완전히 떼면 차량은 스스로 감속하여 멈추며, **완전 정차 시 자동으로 오토홀드(H 아이콘)**가 걸립니다. 즉, 정차 상태를 유지하기 위해 브레이크를 다시 밟을 필요가 없습니다.
* **브레이크 개입:** 테슬라의 회생 제동력은 일정합니다. 앞차와의 간격이 급격히 줄어들거나 급정거가 필요한 상황에서는 망설임 없이 물리 브레이크 페달을 직접 강하게 밟아야 합니다.
* **운전 팁:** 아이오닉 5의 i-Pedal보다 제동이 강하게 느껴질 수 있습니다. 페달을 한 번에 확 떼지 말고 조금씩 되돌리며 감속도를 조절하세요.

## 🤖 오토파일럿 (Autopilot) 상세 조작법

* **활성화:** 주행 중 핸들 오른쪽의 스크롤 휠을 클릭합니다. (설정에 따라 1회 또는 2회 연속 클릭)
* **속도 조절:** 오토파일럿 중 핸들 오른쪽 **스크롤 휠을 위(증가)/아래(감소)**로 돌립니다.
* **차간거리 조절:** 핸들 오른쪽 **스크롤 휠을 왼쪽(멀게)/오른쪽(가깝게)**으로 딸깍 소리가 나도록 밉니다.
* **해제:** 브레이크를 밟거나 핸들을 의도적으로 힘주어 돌리면 즉시 해제됩니다.

## 🎡 스티어링 휠 버튼 상세 맵 (하이랜드 전용)

**왼쪽 버튼 뭉치**
* **▲ / ▼ 버튼:** 우측 / 좌측 방향지시등 (레버가 없으므로 위치 숙지 필수)
* **💡 버튼:** 상향등 (하이빔)
* **왼쪽 스크롤 휠:** 볼륨 조절(상하), 이전/다음 곡 이동(좌우), 재생/정지(클릭)

**오른쪽 버튼 뭉치**
* **🌧️ 버튼:** 와이퍼 (짧게 클릭 시 1회 작동, 꾹 누르면 워셔액 분사 및 화면 조절 메뉴 팝업)
* **🎙️ 버튼:** 음성 명령 실행
* **오른쪽 스크롤 휠:** 오토파일럿 설정 속도(상하) 및 차간거리(좌우) 조절

## ✨ 테슬라 전용 특별 기능 (Useful Tips)

* **화면 재부팅:** 시스템 오류나 통신 장애 시 핸들 양쪽 스크롤 휠 2개를 동시에 10초간 꾹 누르세요. 테슬라 로고가 뜨며 리셋됩니다.
* **감시 모드 (Sentry Mode):** 주차 중 주변 움직임을 녹화합니다. 블랙박스 역할을 하며, 화면 상단 과녁 아이콘으로 설정합니다.
* **도어 열기(안/밖):** 밖에서는 핸들 넓은 쪽을 누르고 당기며, 안에서는 손잡이 위 전자식 버튼을 누릅니다.
* **프렁크 개입:** 앞 트렁크는 외부 버튼이 없으므로 반드시 차량 내부 화면의 컨트롤 메뉴에서 열어야 합니다.
